# NoteShare 
Курсовая работа Дмитриев 4.405-2, 2026.

Совместное приложение для создания и синхронизации заметок в реальном времени. Vue 3 + FastAPI.

## Быстрый запуск

Весь проект (и фронтенд, и бэкенд) запускается одной командой из корневой директории.

### Требования
* **Node.js** (версия 18 или новее)
* **Python** (версия 3.10 или новее)

### Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/qwizzit/NoteShareKurs
   cd NoteShareKurs

2. **Настройте бэкэнд (FastAPI):**
   * Перейдите в папку бэкенда:
     ```bash
     cd backend
     ```
   * Создайте виртуальное окружение:
     ```bash
     py -m venv venv
     ```
   * Активируйте окружение и установите зависимости:
     * *Windows (PowerShell):*
       ```powershell
       .\venv\Scripts\Activate.ps1
       pip install -r requirements.txt
       ```
   * Создайте файл `.env` в папке `backend/` и заполните обязательные ключи конфигурации (`recaptcha_secret_key`, `mail_password` и др.).
   * Вернитесь в корень проекта:
     ```bash
     cd ..
     ```

3. **Настройте и запустите фронтенд (Vue 3):**
   * Установите зависимости (если в PowerShell возникает ошибка безопасности PSSecurityException, выполните команду `npm install` через стандартную Командную строку Windows (cmd), предварительно перейдя в корень проекта):
     ```bash
     npm install
     ```
   * Запустите проект в режиме разработки (одновременно поднимутся фронтенд и бэкэнд):
     ```bash
     npm run dev
     ```
   * После запуска фронтенд будет доступен в браузере по адресу:
     ```text
     http://localhost:5173
     ```
