# Private Notes Backend
Backend part for [lock-note](https://github.com/qwizzit/simple-note)

### How to run locally
1. Clone the repository: `git clone https://github.com/qwizzit/notes-backend.git`
2. Create and activate venv: `python -m venv .venv` и `.venv\Scripts\activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the server: `uvicorn app.main:app --reload`
